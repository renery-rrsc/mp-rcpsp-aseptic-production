# mp-rcpsp-aseptic-production
A proposal of a hybrid improved teaching-learning based optizimation algorithm (TLBO) with greedy randomized adaptative search procedure (GRASP) and serial schedule generation scheme (SSGS) for a multi-objective combinatorial optimization problem of maintenance schedules minimization of total downtime and technicians load variance mathematically modelled as a resource-constrained project scheduling problem (RCPSP) on the scenario of a production line composed of several machines with limited spatial capacity.
